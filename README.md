# deaddoctors-hackday

## Technology Used
- Spring Boot 1.2.7
- Java 8
- Spring 4.1.8
- Bootstrap
- Elastic Search
- Logstash


## How to run locally
1. Clone git repository
2. From command line, Go to the local folder where you clone the repository
3. Run 
```
gradlew build && java -jar build/libs/deaddoctors-hackday-1.0.jar
```
4. Open in browser [http://localhost:8080]
